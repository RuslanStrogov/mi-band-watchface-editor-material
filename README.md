# ⌚ Mi Band Watchface Editor (Material)

Angular Mi Band Watchface Editor — веб-приложение для создания циферблатов с Material Design.

## Начало работы

```bash
git clone git@github.com:RuslanStrogov/mi-band-watchface-editor-material.git
cd mi-band-watchface-editor-material
npm install
```

## Использование

Запустите dev-сервер:

```bash
npm start
```

## Технологии

- **Angular** — фронтенд фреймворк
- **Angular Material** — UI компоненты
- **TypeScript** — типизация
